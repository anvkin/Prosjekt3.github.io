Heim Chat Plan Datasikkerhet

Til venstre kan du sjå databasen til korleis ein løyser problem i eit sosialt medium

    Problem/brukar relasjon
  • Ein brukar kan ha 0 til mange problem
  • Eit problem kan ha minimum 1 maksimum 1 brukar
    Problem/løysing relasjon
  • Eit problem kan ha minimum 0 maksimum 1 løysing
  • Ei løysing kan ha minimum 1 maksimum mange problem
    Løysing/Ansatt relasjon
  • Ei løysing kan ha minimum 1 maksimum mange ansatte som jobber med å fikse det
  • Ein ansatt kan ha minimum 1 maksimum mange løysingar
